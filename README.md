# Round Robin Payout Numerical Simulations

This repository contains the numerical simulation files for the round robin payout study.

Each team should clone this repository and add their own method and all relevant files within the existing folder structure, so that their model and results can be reproduced by others.

## Repository structure

- **Shared/**  
  Contains the description of the common application under study.

- **FE-GCDP-model/**  
  Contains the finite element model used for the finite element analyses with the GCDP model.

## Instructions for contributors

Each team should:

1. Clone the repository.
2. Use the existing folder structure.
3. Add a dedicated folder for their method, including all relevant input files, scripts, documentation, and results needed to reproduce their model.
4. Keep file names and folder organization as clear and consistent as possible.

## Goal

The aim of this repository is to provide a common and reproducible framework for comparing the different numerical approaches used by the participating teams.
